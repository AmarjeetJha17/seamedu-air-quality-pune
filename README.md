# Pune Air Quality & Public Health Crisis
**Seamedu Awards 2026 – Data Analytics Excellence Award**

Data-driven analysis of Pune’s AQI (2017–2024) with pollutant correlations, statistical tests, linear regression (PM2.5 → health risk), and an interactive Streamlit dashboard.

### Key Insights
- Winter AQI is statistically significantly higher than summer (p < 0.001)
- Strongest correlations: PM2.5 ↔ PM10 (0.92), PM2.5 ↔ CO (0.78)
- Linear regression model quantifies health risk from PM2.5
